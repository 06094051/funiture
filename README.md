# funiture

## 前台 - 只是简单的封装展示
* 帮哥们搭一个主要供展示产品的网站
* 项目名称因此而来
* bootstrap 模板
* 相对简单，打算做个后台，维护前台展示数据

## 后台 - 做一些技术的尝试、练手
* 通用的Spring框架搭建
* Maven管理jar包
* 通用的权限管理系统，User - Role - Acl
* 验证码生成及校验
* Redis等缓存使用, 控制台
* 动态Config配置
* Logback + Email通知
* Mybatis + SQL监控
* 线程池
* JSON 序列化与反序列化
* 通用邮件配置及发送
* 文件上传与管理，MD5计算
* HttpClient封装
* Cookie管理
* Excel XSSFWorkbook使用
* 各种Lombok
* Quartz 定时任务
* Hibernate Validator校验
* ThreadLocal管理进程信息
* Junit测试
* 队列rabbitMQ
* Zookeeper Client
